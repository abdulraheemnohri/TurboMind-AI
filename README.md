# TurboMind AI

## 100% Offline Android AI Assistant

**TurboMind AI** is a completely offline AI assistant that runs without internet.

---

## Features

- AI Chat
- Documents AI
- Voice Assistant
- Image Understanding
- Knowledge Base
- Semantic Search
- Offline OCR
- Offline RAG

---

## Getting Started

```bash
git clone https://github.com/abdulraheemnohri/TurboMind-AI.git
cd TurboMind-AI
pip install -r requirements.txt
python main.py
```
