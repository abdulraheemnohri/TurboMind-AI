# TurboMind AI

## 100% Offline Android AI Assistant

TurboMind AI is a completely offline AI assistant that runs without internet. All your data stays on your device - no cloud, no tracking, no API keys.

---

## Features

AI Chat - Text-based AI assistant for all your queries
Documents - Process PDF, DOCX, TXT files
Voice Assistant - Speak and get AI responses
Images - OCR and vision AI for image analysis
Knowledge - Knowledge base with semantic search
Models - Load, unload, download, and benchmark AI models
Settings - Application settings, privacy, theme, model configurations
Search - Hybrid semantic and keyword search

---

## Getting Started

### Desktop

```bash
git clone https://github.com/abdulraheemnohri/TurboMind-AI.git
cd TurboMind-AI
pip install -r requirements.txt
python main.py
```

### Android (Using Buildozer)

```bash
pip install buildozer
buildozer android debug deploy run
```

## Technology Stack

UI Framework: Kivy + KivyMD
AI/ML: PyTorch + Transformers
Offline: 100% Local Processing
Platform: Android + Desktop
Language: Python 3.8+
