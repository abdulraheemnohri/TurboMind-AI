Fonts for TurboMind AI
====================

Required fonts:
- Plus Jakarta Sans (Regular, Bold)

Download from: https://github.com/GoogleFonts/plusjakartasans

To add fonts:
1. Download PlusJakartaSans-Regular.ttf and PlusJakartaSans-Bold.ttf
2. Place them in this directory
3. Update buildozer.spec to include these fonts
